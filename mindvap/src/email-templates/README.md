# Email Templates for MindVap

This directory contains all email templates for the MindVap e-commerce platform. Each template is designed to be responsive, branded, and GDPR-compliant.

## Template Types

### Authentication Templates
- `welcome.html` - Welcome email for new registrations
- `email-verification.html` - Email verification requests
- `password-reset.html` - Password reset emails
- `password-changed.html` - Password change confirmation

### Order Templates
- `order-confirmation.html` - Order confirmation emails
- `order-shipped.html` - Order shipping notifications
- `order-delivered.html` - Order delivery confirmations
- `order-cancelled.html` - Order cancellation notifications
- `order-refunded.html` - Refund notifications

### Security Templates
- `login-alert.html` - New device login alerts
- `account-locked.html` - Account security lock notifications
- `suspicious-activity.html` - Suspicious activity alerts

### Compliance Templates
- `gdpr-consent.html` - GDPR consent requests
- `data-export.html` - Data export confirmations
- `account-deletion.html` - Account deletion confirmations

### Marketing Templates
- `newsletter.html` - Newsletter and promotional emails
- `product-launch.html` - New product announcements
- `special-offer.html` - Special offers and discounts

## Template Features

- Responsive design for mobile and desktop
- Consistent branding with MindVap
- GDPR-compliant unsubscribe links
- Dark/light mode support
- Multi-language support ready
- Accessibility compliant
- Trackable links and images

## Usage

Templates are rendered using the EmailTemplateService and can be customized with dynamic data. Each template includes both HTML and plain text versions.
